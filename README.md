# MintDEV Hub

Welcome to MintDEV Hub, a personal workspace for creative projects, minigames, and more.

## Overview

This project serves as a central hub for:
- **Art Gallery**: Showcasing creative works.
- **Blog**: Sharing thoughts and updates.
- **Tools**: Useful utilities.
- **Minigames**: Small, fun games.
- **ARG**: Hidden secrets and puzzles.

## Setup

To set up the project environment (Termux/Linux):

```bash
bash setup.sh
```

## Maintenance

To update the project and run migrations:

```bash
bash maintenance.sh
```
