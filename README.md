## Setup

To install and set up the project on Termux/Linux with a single command:

```bash
curl -sL https://raw.githubusercontent.com/mereMint/o7MintDOTdev/main/src/scripts/bash/setup.sh | bash
```

This script will:
1. Install necessary dependencies (Node.js, Git, MariaDB).
2. Clone the repository to `~/MintDEV`.
3. Set up the database.
4. Install Node.js dependencies.
